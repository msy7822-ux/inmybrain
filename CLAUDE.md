# in_brain - Research Knowledge Vault

## 概要
このリポジトリはClaude Codeと一緒にリサーチした知識を蓄積する場所です。

## 構造
```
in_brain/
├── topics/           # トピック別リサーチノート
│   ├── ai-agents/    # AIエージェント関連
│   ├── dev-tools/    # 開発ツール関連
│   └── ...
├── templates/        # テンプレート
└── README.md         # インデックス
```

## リサーチノートの形式

各リサーチノートは以下の形式で記述：

```markdown
# タイトル

## メタデータ
- **調査日**: YYYY-MM-DD
- **ソース**: [URL or 出典]
- **タグ**: #tag1 #tag2

## 概要
1-2文での要約

## 詳細
本文

## キーインサイト
- ポイント1
- ポイント2

## 関連
- [[関連トピック]]
```

## Claude Codeへの指示

1. **新規リサーチ保存時**
   - `topics/` 配下の適切なカテゴリに保存
   - テンプレートに従って記述
   - README.mdのインデックスを更新

2. **既存知識の参照時**
   - まず `topics/` 内を検索
   - 関連するノートがあれば参照して回答に活用

3. **カテゴリがない場合**
   - 新しいフォルダを作成してOK
   - README.mdに追記
